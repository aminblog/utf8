PCRE Regular expressions for UTF-8.

Note this file is not actually used by the rest of the library but these regular expressions can be useful to have available. The regular expressions are modified to include full ASCII range including control chars.

  • see: http://www.w3.org/International/questions/qa-forms-utf-8
