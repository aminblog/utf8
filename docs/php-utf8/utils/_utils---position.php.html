utf8_byte_position (line 25)

Given a string and a character index in the string, in terms of the UTF-8 character position, returns the byte index of that character.

Can be useful when you want to PHP's native string functions but we warned, locating the byte can be expensive. Takes variable number of parameters - first must be the search string then

  1. to n UTF-8 character positions to obtain byte indexes for - it is more
efficient to search the string for multiple characters at once, than make repeated calls to this function

  • return: - int if only one input int, array if more, boolean TRUE if it's all ASCII

utf8_locate_current_chr (line 130)

Given a string and any byte index, returns the byte index of the start of the current UTF-8 character, relative to supplied position.

If the current character begins at the same place as the supplied byte index, that byte index will be returned. Otherwise this function will step backwards, looking for the index where curent UTF-8 character begins.

  • return: byte index of start of next UTF-8 character
  • author: Chris Smith<chris@jalakai.co.uk>
